Config

public struct Config

A configuration object for Client.

To create a new Config object a non-empty API key string has to be provided. After initializing, it’s possible to change additional parameters:

    var config = Client.Config(apiKey: "<API_KEY>")
    config.baseURL = .dublin
    config.logOptions = [.debug]
