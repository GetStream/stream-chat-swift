A `NavigationRouter` subclass that handles navigation actions of `ChatChannelListVC`.

``` swift
open class _ChatChannelListRouter<ExtraData: ExtraDataTypes>:
    NavigationRouter<_ChatChannelListVC<ExtraData>>,
    ComponentsProvider
```
