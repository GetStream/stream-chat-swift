### `attachment(payloadType:)`

Converts type-erased attachment to the attachment with the concrete payload.

``` swift
func attachment<Payload: AttachmentPayload>(
        payloadType: Payload.Type
    ) -> ChatMessageAttachment<Payload>? 
```

#### Parameters

#### Returns
